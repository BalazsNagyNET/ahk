# ahk

AutoHotkey files.

- `v1.1/` — original AutoHotkey v1 scripts (classic RDP via .rdp file, mstsc)
- `v2.0/` — AutoHotkey v2 scripts, targeting Windows App remote sessions (msrdc.exe)
  - `ConnectWork.ahk` — Ctrl+Alt+W: FortiClient VPN + open workstation (see `v1.1/SETUP.md`)
  - `RDPHotkeyHelper.ahk` — media/volume keys stay local inside the remote session
  - `change_playback_device.ahk` — Win+A cycles local playback device, also inside the session
  - `minimize_activate_shift_caps.ahk` — Shift+CapsLock minimize/restore toggle
